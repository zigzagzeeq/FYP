# Predictive Modelling of Anomalous Driver Behaviour Using GPS Telemetry for Logistics Fleet Security

This research develops a predictive analytics system for real-time driver behaviour monitoring and risk assessment in commercial logistics fleets, leveraging high-frequency GPS telemetry streams. A multi-stage pipeline is designed encompassing signal preprocessing, feature engineering (kinematic, spatial, and temporal features), and ensemble classification to capture sequential trip patterns. The model distinguishes routine operational variance from anomalous behaviours associated with fuel theft, route deviation, unauthorised vehicle use, and aggressive driving. An online learning module incrementally updates the model as new labelled events are confirmed by fleet operators. The system is deployed and evaluated across a 12-month pilot at two logistics depots in Peninsular Malaysia, measuring precision, recall, false-positive rate, and operational impact — targeting ≥90% anomaly detection precision and a verified ≥20% reduction in diesel loss and theft incidents.
